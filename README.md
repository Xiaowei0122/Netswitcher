# 网络一键切换器

> 一款用于快速切换和配置网络设置的实用工具。

## 📌 项目简介

**网络一键切换器** 是一款便捷的网络配置工具，旨在帮助用户一键切换不同网络环境，简化手动修改网络参数的繁琐流程。适用于经常在办公网络、家庭网络、移动热点等多种网络环境中切换的用户。

## ✨ 核心功能

- 一键切换多个预设的网络配置（如 IP、DNS、网关等）
- 快速应用新的网络设置
- 支持静态 IP 和 DHCP 模式配置
- 支持导入/导出配置方案
- 简洁易用的图形界面（如适用）
- 多平台支持（如适用）
